# Métricas e fórmulas do funil

Definições para coletar e calcular de forma consistente. Use a mesma linguagem do
cliente quando ele tiver nomes próprios para as etapas.

## Topo (marketing)
- **Impressões** — quantas vezes os anúncios apareceram.
- **Cliques** — cliques no anúncio.
- **CTR** = cliques / impressões × 100.
- **Investido (spend)** — gasto de mídia no período.
- **Leads** — formulários/conversões de lead.
- **CPL** = investido / leads (custo por lead).
- **Frequency** — média de vezes que a mesma pessoa viu (alto = fadiga de criativo).

## Handoff
- **MQL** — lead qualificado de marketing (apto a abordagem). Classificação
  comercial, **não** vem do Meta.
- **Taxa de MQL** = MQL / leads × 100 (qualidade do lead).
- **CPMQL** = investido / MQL (custo por MQL — ótimo p/ ver semana que piorou).

## Fundo (comercial)
- **Taxa de agendamento** = agendaram / MQL × 100. Referência boa de pré-venda
  estruturada: 30–50%.
- **No-show** = agendados que não compareceram.
- **Taxa de proposta** = propostas / R1.
- **Fechamento** = contratos / propostas (ou / R1).
- **Pipeline ativo** — deals em aberto (não somam como venda ainda).

## Resultado (quando houver valor)
- **Receita** = soma dos contratos fechados.
- **ROAS** = receita / investido. **CAC** = investido / clientes.
- **Ticket** = valor médio por contrato.

> Sempre marque o **ponto de maior vazamento** (a etapa com pior conversão relativa)
> — é o que orienta o plano de ação.
